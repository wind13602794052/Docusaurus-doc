# 计算公式

## 当前订单价格

- 当前订单价格 = 订单全部商品重新加入购物车价格-用户使用的积分-用户使用的coupon


## 商品换货
- 当前订单价格 = 更换商品sku+旧sku, 重新加入购物车价格-用户使用的积分-用户使用的coupon

## 使用coupon

- 判断新订单数据是否适用coupon，暂时人工判断
